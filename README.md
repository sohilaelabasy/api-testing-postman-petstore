## Overview
This is a practice project for REST API testing using Postman on the Swagger Petstore API.
The goal is to validate API functionality and learn writing automated assertions using Postman Tests.

## What I Tested
- HTTP status code validation
- Response body verification (JSON)
- Header validation
- Response time checks
- Positive scenarios (CRUD basics)
- Automated assertions using Postman Tests (JavaScript)

## Tools & Skills
- API Testing
- Postman
- REST
- JSON
- JavaScript Assertions

## Project Files
- Postman Collection: `Petstore.postman_collection.json`
- Postman Environment: `Petstore.postman_environment.json` (if included)

## How to Run
1. Open Postman
2. Import the collection JSON file
3. Import the environment file and select it
4. Run the collection using **Collection Runner**
5. Review the test results (Pass/Fail)
